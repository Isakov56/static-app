CENTRAL ASIA TOURS - QUICK START GUIDE
================================================================================

📋 SUMMARY:
   This website uses an object-wrapper technique to resist Firefox text-only
   zoom while maintaining all original styling and functionality.

🎯 TO OPEN THE WEBSITE:
   
   Open this file:  index-wrapper.html
   
   (NOT index.html or index-content.html)

🧪 TO TEST FIREFOX TEXT-ONLY ZOOM:

   1. Open Firefox browser
   2. Type in address bar: about:preferences
   3. Scroll to "Language and Appearance" section
   4. Find "Zoom" settings
   5. CHECK the box: "Zoom text only"
   6. Open index-wrapper.html in Firefox
   7. Press Ctrl+ or Ctrl- to zoom
   8. ✓ Text should NOT change size (stays static)

🧪 TO TEST REGULAR ZOOM:

   1. Open index-wrapper.html in any browser
   2. Press Ctrl+ or Ctrl- to zoom
   3. ✓ Page should scale proportionally to fit viewport

🧪 TO TEST WINDOWS SCALING:

   1. Windows Settings → System → Display
   2. Change "Scale" to 125%, 150%, or 200%
   3. Open index-wrapper.html
   4. ✓ Content should display at consistent size

📁 FILE STRUCTURE:

   index-wrapper.html  ← MAIN FILE (open this)
   index-content.html  ← Content (loaded by wrapper)
   index.html          ← Original (reference only)
   styles.css          ← Styles
   script.js           ← JavaScript
   README.md           ← Full documentation

✅ ASSIGNMENT COMPLIANCE:

   ✓ Pure HTML/CSS/JavaScript (no frameworks/libraries)
   ✓ Static design (no visual changes during zoom)
   ✓ Windows DPI scaling support
   ✓ Firefox text-only zoom resistance
   ✓ 1920px fixed width layout

⚠️  TROUBLESHOOTING:

   Problem: Text still zooms in Firefox
   Solution: Make sure you opened index-wrapper.html (not index.html)
   
   Problem: Content not showing
   Solution: Ensure all files are in the same folder
   
   Problem: Scaling issues
   Solution: Check JavaScript is enabled in browser
